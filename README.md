# OMANEKUN


## 概要
 
自動車の整備依頼データを管理するアプリです。依頼内容をDB上に保存しておくことで、
検索機能により目当てのデータを探す手間なく得ることができます。
サーバーサイドの処理に関してはPHP言語単体で、フレームワークは使わずに開発しました。
 

***デモ***
 
https://kum634.com/omanekun/
 

## 機能
 
- データベースへのデータの挿入、取得、削除。
- キーワード等で任意の条件を指定して検索し、一致したものをデータベースから取得。
- 挿入したデータの日付が今週に当てはまるもののみ、データベースから自動的に取得し表示。
- 取得した値から自動的に作業指示書を作成。
- データベースでユーザー情報を管理（新規登録、ログイン、自動ログイン、パスワード再設定メール、退会機能）。
- CSV入出力機能。
- Bootstrapによるレスポンシブ対応。
